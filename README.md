# CarPriceRegression

### Struktura analizy
1. Zależności między zmiennymi (liczbowe + kategoryczne)
2. Zakres i zmienność (statystyki opisowe, rozkłady, outliery)
3. Stopień wypełnienia danych (braki i ich interpretacja)
4. Wizualizacje wspierające wnioski
5. Raport końcowy + hipoteza badawcza pod regresję ceny
